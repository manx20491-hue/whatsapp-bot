# WhatsApp Bot 🤖

This is a simple WhatsApp bot built using Baileys.

## Features
- Auto reply
- Simple commands
- Connection via QR scan

## Commands
- hi -> greeting
- menu -> show commands
- ping -> test bot

## Install
```
npm install
```

## Run
```
npm start
```

## Setup
1. Run bot
2. Scan QR code
3. Start chatting
